<div style="text-align:center;font-weight:bold;color:green;font-size:150%">Bliki wurde erfolgreich installiert!</div>

Sie können jede Seite bearbeiten, in dem Sie den Reiter "Seite bearbeiten" oberhalb eines Textes anklicken.
Konfigurationseinstellungen für dieses Wiki können auf der Spezialseite [[Special:Admin]] angepasst werden. 
Zusätzlich können die folgenden Seiten geändert werden, um das Aussehen Ihres Wikis individuell anzupassen:

* [[LeftMenu]] - Das links auf jeder Seite vorhandene Navigationsmenü.
* [[BottomArea]] - Die Fußzeile die unten auf jeder Seite erscheint.
* [[StyleSheet]] - Das Stylesheet das CSS Konfigurationseinstellungen zur grafischen Gestaltung dieser Webseiten enthält.

Bitte teilen Sie uns auf [http://code.google.com/p/gwtwiki/ Bliki Project] mit, ob Ihre Bliki Installation erfolgreich war. 

Einige weitere nützliche Funktionen:

* [[Special:RecentChanges]] - Eine Liste der zuletzt geänderten Wiki Artikel.
* [[Special:Upload]] - Spezialseite zum Hochladen von Dateien und Bildern in das Wiki benutzt.
* [[Special:Search]] - Spezialseite zum Durchsuchen des Wikis.
* [[Special:Account]] - Legen Sie ein Benutzerkonto an, so dass Änderungen an den Wiki Artikeln Ihrem Benutzer entsprechend zugeordnet werden können.
* [[Special:Allpages]] - Eine Liste aller Wiki Artikel.
* [[Special:Specialpages]] - Ein Index der vorhandenen Spezialseiten.
